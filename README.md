
# استوديو لافندر - تصوير فعاليات الكويت (Lavender Studios)

تم تحديث الموقع بصور فوتوغرافية حقيقية وتصحيح أيقونات التواصل الاجتماعي مع تحسين واجهة "الهيرو".

## التنبيهات الأمنية والخصوصية
- **قاعدة صفر نساء**: يمنع منعاً باتاً استخدام صور تظهر فيها النساء أو ملامحهن، التزاماً بهوية الاستوديو في تغطية المناسبات الرسمية والرجالية في الكويت.
- **إدارة الصور**: الصور الحالية تستخدم روابط Unsplash (لأغراض العرض الفوري). **يجب** تحميل هذه الصور وحفظها محلياً في المسار `/public/assets/photos/` وتحديث الروابط في ملفات `content/*.ts` لضمان العمل بدون إنترنت.

## التقنيات المحدثة
- **Hero Overlay**: تم تحسين التباين والقوة البصرية باستخدام Gradients مركبة.
- **Social Icons**: تم استخدام أيقونات SVG دقيقة لـ (Instagram, X, LinkedIn) في الفوتر.

## كيفية التشغيل
1. تثبيت المكتبات:
   ```bash
   npm install
   ```
2. تشغيل بيئة التطوير:
   ```bash
   npm run dev
   ```

## إضافة أعمال جديدة
لتعديل معرض الأعمال دون كسر التصميم:
1. أضف الصورة لمجلد `/public/assets/photos/`.
2. افتح `/src/content/work.ts` وأضف عنصراً جديداً مع تحديد الفئة (Wedding, Corporate, etc).
3. التصميم يعتمد نظام Grid مرن (Masonry-like) سيتكيف تلقائياً مع حجم الصورة.
